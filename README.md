# 📝 Tasky - Task Management App

A modern and interactive task management application built using HTML, CSS, and JavaScript to help users efficiently manage daily tasks.

🔗 Live Demo: https://kaushalhole24.github.io/Tasky-App/

---

## 🚀 Features

- ➕ Add new tasks easily
- ✏️ Edit existing tasks
- ❌ Delete tasks with smooth experience
- 🔍 Filter tasks (All / Completed / Pending)
- 📊 Real-time task statistics (Total, Completed, Remaining)
- 📈 Animated progress tracking
- 💾 Data persistence using localStorage
- ⚡ Responsive and clean UI design

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)

---

## 🎯 How to Use

1. Enter a task in the input field
2. Click **Add Task** or press Enter
3. Mark tasks as completed using checkbox
4. Edit tasks using edit icon
5. Delete tasks using delete icon
6. Use filters to view specific tasks
7. Track your progress using the progress circle
